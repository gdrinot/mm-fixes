# mm-fixes
Fixes investment transactions

First create a tag on Monarch to tag transactions already fixed
Inspect item and use network, tag any transaction and take note of the ID.
Paste the ID into the [INSERT_TAG_ID]
Run the code on the console and start scrolling through the investment transactions.
This will fix them and note any tickers missing from the dictionary. Add them and run again.

The fixed transaction will be:
Merchant: "TICKER" or "Crypto: TICKER" or "TICKER Call/Put"
Note: "Stock/Crypto/Call PRICE" if available
